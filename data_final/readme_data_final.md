# Final Data README

This folder stores final analysis-ready datasets.

Final data are not raw source files and are not casual intermediate checkpoints. They should be cleaned, merged, harmonized, documented, and ready for approved empirical analysis.

## Core Rule

Every final dataset must be fully reproducible from raw and/or intermediate data using documented project code.

Do not directly edit final datasets by hand.

If values, variables, labels, sample restrictions, merge rules, or file structures need to change, update the code that generates the final dataset and regenerate it.

## Required Metadata for Each Final Dataset

Document:

- file name and path;
- generating script;
- source inputs;
- unit of observation;
- key identifiers;
- row count;
- sample restrictions;
- constructed variables;
- intended analysis use;
- known limitations.
